HTML

Todos los comandos que se mencionaran deben de llevar su prefijo "<>" al iniciar y al terminar un "/" antes del comando.

Estructura básica de un documento

Comandos Descripción
!DOCTYPE Esta tag de HTML determina el tipo de documento.
html Con esta etiqueta, el documento se definirá como documento HTML.
head En la sección head se almacenan los metadatos del documento.
title Este comando contiene el título del documento, que además se mostrará en la barra de título del buscador.
body body es la sección principal y engloba todo el contenido que verás en el navegador.
nav La sección de navegación de una web
section Con section puedes resumir los elementos en un grupo.
article article es la sección de contenido de una web.
header header define el encabezado de una web o una sección.
footer footer define el pie de página de una web o una sección.

Ordenar una web

Comandos Descripción
h1 a h6 Los titulares van en las distintas tags h. Cuanto menor sea la cifra, mayor será el encabezado.
p indica que se trata de un párrafo.
br Con la tag HTML br se fuerza un salto de línea
hr crea una línea divisoria física entre dos tipos de contenido.

Tipo de letra

Comandos Descripción
b Poner en negrita palabras sueltas, frases o párrafos.
i Poner en cursiva palabras sueltas, frases o párrafos.
u El fragmento seleccionado estará subrayado.
s El texto seleccionado estará tachado.
sup el carácter seleccionado será un superíndice, p. ej. 1o.
sub puedes escribir caracteres en subíndices, como el 2 de H2O.

Resaltar y marcar párrafos

Comandos Descripción
strong para resaltar párrafos.
em puedes enfatizar fragmentos de texto.
q puedes marcar citas dentro de un texto.
blockquote
marca todo un párrafo como cita

Crear tablas y listas

Comandos Descripción
table creas una tabla.
caption define el título de la tabla.
tr Las líneas de la tabla se etiquetan
td define una casilla de la tabla.
th La celda del encabezado de una tabla se define
ol crea una lista con números ordinales.
ul crea una lista con viñetas.
li indica cada punto de la lista.
di indica una lista de definiciones.
dt define un concepto o posición en la lista de definiciones.
dd define un concepto o posición en la lista de definiciones.

Contenido multimedia

Comandos Descripción

map puedes insertar un mapa en un documento.
audio te permite insertar un sonido.
video puedes añadir contenido en vídeo.

Formularios

Comandos Descripción

input se utiliza para controlar lo que se escribe.
button se utiliza para controlar lo que se escribe.
select te permite crear una lista de opciones.

Inscrustar enlaces

Comandos Descripción

Otros comandos

Comandos Descripción
